Projects

MY_LS

I reimplemented the ls command in C. The project involved reading directories, handling file paths, and displaying file information in different formats. It gave me practical experience with system calls, directory traversal, and string formatting./p>

MY_TOP

I created a simplified version of the top command. It required retrieving process information, updating it in real time, and presenting it in a clear layout. Through this, I gained insights into process monitoring, memory usage, and performance statistics.

MY_SUDO

I developed a minimal version of sudo, focusing on permission handling and privilege escalation. This project improved my understanding of user authentication, system security concepts, and access control.

MY_HUNTER A.K.A DUCK HUNT

A remake of the classic Duck Hunt game written in C. Focused on handling user input, event-driven programming, and collision detection within a continuous game loop./p>

MY_RADAR

Developed a radar simulation where radars detect airplanes within a base radius. Airplanes inside the safe zone avoid collisions, while those outside may collide and be destroyed. Practiced real-time object tracking and collision handling logic.

SETTING_UP

Built a program that generates a map with obstacles and finds the largest possible square that fits without overlap. Strengthened algorithmic problem-solving skills and 2D array manipulation.

ORGANIZED

Created a data management program with functions like add, delete, and sort, implemented using linked lists. Improved knowledge of data structures, dynamic memory allocation, and pointer operations in C.

SECURED

Implemented a hashing utility that stores words in a table for retrieval. This project introduced me to hash functions, hash tables, and efficient storage techniques for quick access.

MY_PRINTF

I reimplemented the printf function in C. It required parsing format specifiers, managing variable arguments, and producing correctly formatted output. This helped me strengthen my knowledge of variadic functions, memory management, and string manipulation.
